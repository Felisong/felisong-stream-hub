# Welcome to Felisong's Stream Hub

I'm Felisong!

I am working on developing a cat spawner for my stream. This is a little project I just wanted to get started on. It will be something like Shimeji's were if you were in the know back in around 2013. It's when you could have custom sprites of characters or animals walk around and interact with your screen. I never made a custom one but had it installed and really loved them. I was inspired to make a cat version.

I will be using what i know HTML, CSS, JS to create it and hopefully aim to make it as lightweight as possible. This project may change with time as I am amking it custom to me, but plan to have a fork with just the basic cat spawner to allow for free use. Something fun about making things.
